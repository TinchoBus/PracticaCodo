Definición

Márgenes (margin): Espacio en blanco alrededor de un elemento, separándolo de otros elementos.
Relleno (padding): Espacio en blanco dentro de un elemento, entre el contenido y el borde del elemento.
Bordes (border): Líneas que rodean un elemento, separándolo visualmente de otros elementos.
background color: Color de fondo de un elemento.
width: Ancho de un elemento.
height: Altura de un elemento.
display: Propiedad que define cómo se muestra un elemento en la página (por ejemplo, bloque, en línea, etc.).
flexbox: Modelo de diseño que permite organizar elementos en una sola dimensión (horizontal o vertical) de manera flexible.
grid: Modelo de diseño que permite organizar elementos en una cuadrícula bidimensional, facilitando la alineación y distribución de los mismos.
position: Propiedad que define cómo se posiciona un elemento en la página (por ejemplo, estático, relativo, absoluto, fijo).
overflow: Propiedad que controla cómo se muestra el contenido que excede el tamaño de un elemento (por ejemplo, ocultar, desplazar, etc.).
float: Propiedad que permite que un elemento se desplace a la izquierda o derecha dentro de su contenedor, permitiendo que otros elementos fluyan a su alrededor.
clear: Propiedad que controla el comportamiento de los elementos flotantes, evitando que otros elementos se coloquen a su lado.
